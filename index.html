How to Generate Thread Dumps

Generate a JSON thread dump from your Java application using:

jstack -format=json <pid> > thread-dump.json

Or with JDK 21+ and virtual threads:

jcmd <pid> Thread.dump_to_file -format=json thread-dump.json

About Threadly

Threadly is a free, open-source Java thread dump analyzer that helps developers debug concurrency issues, detect deadlocks, and analyze thread performance. All processing happens client-side for maximum privacy and security.

  • Thread state visualization
  • Performance insights and recommendations
  • Interactive thread dump exploration
